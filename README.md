# Audion - Music Player

ECE 150 - Project

## Hardware

1. Onion Omega Board
2. Bluetooth Speaker
3. Push Buttons (Play | Pause)
4. Wiring set up on Breadboard

## Software 

Written in C++, we create the following actions: 
1. Read WAV files from the ALSA libary.
2. Play music
3. Pause music
4. Navigate the system
5. Read GPIO pins
6. Logging
